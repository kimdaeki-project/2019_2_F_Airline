
# Airline
> 아시아나에 기반한 항공권 예매 사이트 제작 
- 개발기간 : 2020.01.13 ~ 2020.02.17

---

## 개발환경

- **운영체제** : Window10 64bit / Linux
- **사용언어** : JAVA(JDK 1.8) / JavaScript / JSP 
- **DB** : MySQL
- **Framework** : Spring Boot / Mybatis / BootStrap
- HTML5 / CSS / jQuery / Ajax / Json
- Git / Maven
- **API**
